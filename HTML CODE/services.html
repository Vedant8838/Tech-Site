Network Swithches/Routers

    Network switches and routers are networking hardware that connect and route data between devices. Switches are used to connect devices on a single network, while routers move data between networks.

  • Also known as Ethernet switches, network switches connect devices on a single Lan
  • They are restricted to wired network connections
  • They are part of the data-link layer of the OSI model
  • They use MAC addresses to transfer data to the correct destination
  • They are less sophisticated than routers
  • They are restricted to wired network connections


      Routers

  • They are primarily Layer 3 devices on the OSI model
  • They move data between two or more computer networks
  • They can work within both wired and Wi-Fi networks
  • They use IP addresses to determine the route and destination of a packet
  • They offer network address translation, NetFlow, and quality of service




    Server-Storage

    Server storage is a place to store data that is not in use. It is important to store data in a permanent solution, such as a storage area network (SAN), cloud storage, network-attached storage (NAS), or direct-attached storage (DAS).

      Types OF Server-Storage

  • Backup server A server that stores important data, files, folders, databases, and hard drives
  • Multimedia server A server that provides services to clients over a series of networks
  • Storage dedicated server A server with enhanced storage capacities for businesses that manage large volumes of data
  • Hybrid storage A system that combines flash-based solid-state drives (SSDs) and mechanical hard-disk drives (HDDs)
